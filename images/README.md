# 图片文件夹

将你的项目图片放在这个文件夹中。

## 建议的图片命名规范：

### 首页缩略图（400x300像素，推荐）
- project1-thumb.jpg
- project2-thumb.jpg
- project3-thumb.jpg
- project4-thumb.jpg
- project5-thumb.jpg
- project6-thumb.jpg

### 详情页主图（900x500像素，推荐）
- project1-main.jpg
- project1-detail1.jpg
- project1-detail2.jpg
- project2-main.jpg
- project2-detail1.jpg
...

## 图片格式建议：
- 使用 JPG 格式（文件小，加载快）
- 或使用 PNG 格式（支持透明背景）
- 或使用 WebP 格式（更小的文件大小）

## 图片尺寸建议：
- 首页缩略图：400x300 像素
- 详情页主图：900x500 像素
- 保持图片清晰但文件大小适中（每张图片建议不超过500KB）
